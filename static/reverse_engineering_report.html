📐Gabor Filter Function

The fundamental building block discovered in learned weights — a Gaussian-windowed sinusoid.

📝 Gabor Filter Definition
G(x; A, μ, σ, f, φ) = A · exp(-(x-μ)² / (2σ²)) · sin(2π · f · x / d_model + φ)

🔑Complete W_k Equation

The learned key projection matrix as a sum of Gabor filters across frequency components.

🔑 Key Weight Matrix W_k
W_k(l,h)[i,j] = Σf∈F(l,h) A_k(l,h)[i,f] · G(j; μ(l,h)[i], σ(l,h)[i], f, φ_k(l,h)[i,f])
📊 Amplitude Distribution
A_k(l,h)[i,f] = A_base(l,h) · exp(-(f - fcenter)² / (2σf²)) + N(0, σA²)
🎯 Center Position
μ(l,h)[i] = dmodel/2 + Δ_μ(l,h) · (2i/dmodel - 1) ≈ 64 ± 5

🔍Complete W_q Equation with Phase Shifts

The query projection matrix shares the Gaussian envelope with W_k but includes a learned phase shift δ.

🔍 Query Weight Matrix W_q
W_q(l,h)[i,j] = Σf∈F(l,h) A_q(l,h)[i,f] · G(j; μ(l,h)[i], σ(l,h)[i], f, φ_k(l,h)[i,f] + δ(l,h)[i,f])
🔄 Phase Shift Relationship
φ_q(l,h)[i,f] = φ_k(l,h)[i,f] + δ(l,h)[i,f]

δ(l,h) = d · 2πf / dmodel + δ_bias(l,h)

For previous token head (d=1, f=32): δ = π/2

⚡Attention Score Equations

⚡ RoPE Rotation
θ_t = 10000-2t/dhead

RoPE(x, m)2t = x2tcos(mθ_t) - x2t+1sin(mθ_t)
RoPE(x, m)2t+1 = x2tsin(mθ_t) + x2t+1cos(mθ_t)
🎯 Attention Score
score(l,h)(m,n) = RoPE(W_qhm, m) · RoPE(W_khn, n) / √dhead

Key Property: Depends only on relative position (m - n)!

🔄Complete Forward Pass

1️⃣ Token Embedding
hi(0) = E[ti] ∈ ℝdmodel
2️⃣ Layer Normalization
LN(x) = γ ⊙ (x - μ) / √(σ² + ε) + β
3️⃣ Multi-Head Attention (Per Layer l)
Q(l,h) = h(l) W_q(l,h)T
K(l,h) = h(l) W_k(l,h)T
V(l,h) = h(l) W_v(l,h)T

A(l,h) = softmax(Q̃(l,h)K̃(l,h)T / √dhead + mask)
o(l,h) = A(l,h)V(l,h)
4️⃣ Residual Connections
h(l+½) = h(l) + Attn(l)(LN(h(l)))
h(l+1) = h(l+½) + FFN(l)(LN(h(l+½)))
5️⃣ Output Projection
zi = Wout · LN(hi(2))
P(token=v | context) = exp(zi,v) / Σv' exp(zi,v')

W_k[i,j] = 1 if j % 2 = 0

Synthesis

Conclusion

Synthesizing the findings into actionable insights about transformer learning dynamics.

Both solutions work but through fundamentally different mechanisms. The hand-coded approach represents a mathematical optimum: discrete, elegant, and theoretically grounded. The SGD-discovered solution, in contrast, emerges as a distributed, multi-resolution, statistically optimal representation.

This reveals a profound insight: the space of induction-capable solutions is much larger than theory captures! Gradient descent doesn't necessarily converge to human-designed mathematical structures — it discovers its own pathways that achieve functional equivalence through entirely different representational strategies.

The Gabor filter structure discovered in learned weights bears striking resemblance to biological receptive fields, suggesting fundamental principles of efficient information processing that transcend both biological and artificial systems.

Key Takeaways

  • 1 SGD prefers continuous, distributed representations over discrete patterns
  • 2 Full-rank solutions emerge even when low-rank would suffice
  • 3 Multiple valid solution basins exist for the same task
  • 4 Theory-guided initialization is not a basin of attraction
  • 5 Phase shifts between W_q and W_k encode attention patterns
